🎬 Netflix Content Analysis Dashboard (Power BI)
-----------------------------------------------------

📌 Project Overview
----------------------

This project presents an interactive data analysis of Netflix content using Power BI.
The dashboard focuses on content distribution, platform growth, and regional insights to understand content strategy and expansion patterns.

⚡ Built with business-focused insights, not just visuals.

🎯 Objectives
---------------
1. Analyze distribution of Movies vs TV Shows

2. Identify production vs platform growth trends

3. Detect peak content expansion periods

4. Understand country-wise contribution

5. Enable deep-dive analysis using drillthrough

📊 Dashboard Structure
---------------------------

🟢 1. Overview
----------------

1. KPIs: Total Titles, Countries, Genres, Total Duration / Seasons

2. Top 10 Countries by content

3. Genre distribution

🔵 2. Movies vs TV Shows Analysis
----------------------------------

1. % of Movies vs TV Shows

2. Average Movie Duration vs Average TV Seasons

3. Country-wise comparison

4. Content type distribution

🟣 3. Trend Analysis
----------------------

1. Titles by Release Year (Production Trend)

2. Titles by Date Added (Platform Growth Trend)

3. Peak Production Year vs Peak Platform Addition Year

4. Time-based content expansion insights

🟠 4. Country-Level Drillthrough Analysis
------------------------------------------

1. Dynamic analysis based on selected country

2. KPIs: Total Titles, Country Contribution %, Peak Year, Avg Movie Duration, Avg TV Seasons

3. Genre drivers (what type of content dominates the country)

4. Movies vs TV Shows preference

5. Audience rating distribution

6. Year-wise content trend with peak highlight

🔍 Users can click on any country and navigate to a dedicated analysis page using the drillthrough feature for deeper insights.

🔥 Key Insights
----------------

1. Movies dominate the platform, indicating scalable content strategy

2. Content is highly concentrated in a few countries

3. Significant growth observed after 2015

4. Platform expansion (Date Added) differs from production trend (Release Year)

5. TV Shows tend to have shorter season spans

🧠 Analytical Approach
------------------------

1. Data Cleaning & Transformation using Power Query

2. DAX Measures & Calculated Columns

3. Time-based comparative analysis

4. KPI-driven dashboard design

5. Drillthrough implementation for detailed insights

🛠️ Tools & Technologies
-------------------------

1. Power BI

2. DAX (Data Analysis Expressions)

3. Power Query

4. Data Modeling
